# markrEdge

A Node.js project for markrEdge application.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation
```bash
npm install
```

### Running the Application
```bash
npm start
```

### Development
```bash
npm run dev
```
